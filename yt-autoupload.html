YouTube API 설정
YouTube Data API v3를 사용해 영상을 자동 업로드합니다.
⚙️
설정 방법:
1. Google Cloud Console → 프로젝트 생성 → YouTube Data API v3 활성화
2. OAuth 2.0 클라이언트 ID 생성 (웹 애플리케이션 유형)
3. 승인된 리디렉션 URI에 현재 페이지 URL 추가
4. 클라이언트 ID를 아래에 입력 후 Google 계정으로 로그인
